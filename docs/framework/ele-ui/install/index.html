# CDN 引入

可通过CDN的方式引入form-build，请确保element-ui在form-build之前引入。form-build会自动注册为全局组件。

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <!-- 引入样式 -->
  <link rel="stylesheet" href="https://unpkg.com/element-ui/lib/theme-chalk/index.css">
</head>
<body>
  <div id="app">
    <form-build :form-list="formList" ref="form" :model="model">
      <template slot="action-button">
        <el-button block type="primary" @click="submit">提交</el-button>
        <el-button block @click="reset">重置</el-button>
      </template>
    </form-build>
  </div>
</body>
  <!-- 引入Vue -->
  <script src="https://unpkg.com/vue/dist/vue.js"></script>
  <!-- 引入组件库 -->
  <script src="https://unpkg.com/element-ui/lib/index.js"></script>
  <!-- TODO 引入 form-build -->
  <script src="//unpkg.com/form-build/dist/form-build.min.js"></script>
  <script>
    new Vue({
      el: '#app',
      data: function() {
        return { 
          model: {},
          formList: [/* your config */]
        }
      }
    })
  </script>
</html>

说明

使用 CDN 引入时，请在链接地址上锁定版本，以免你的项目受到非兼容更新的影响。

# NPM 引入

npm安装插件

npm i form-build -S

在入口文件(main.js)中注册组件：

import Vue from 'vue'
import formBuild from 'form-build'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(formBuild, {
  UIname: 'element-ui',
  UIplugin: ElementUI
})

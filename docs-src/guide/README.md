## form-build
> 让表单使用更简单

form-build数据化配置驱动表单，分别支持pc代表ui库<a href="https://element.eleme.cn" target="_blank">element-ui</a>和移动端代表ui库<a href="" target="_blank">vant</a>，提供数据校验、获取数据模型及自定义组件等特色功能。

## 特色功能

* 数据化配置
* 自定义组件
* 表单间可通信
* 表单label可配置
* 静态化表单
* 表单input/ouput数据处理
* 表单校验
* 栅格布局
